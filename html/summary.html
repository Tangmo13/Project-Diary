Z-index

z-index คือตัวกำหนดจัดการ การซ้อนทับของ layout ในแกน z ยิ่งค่าของ z-index เป็นบวกจะเป็นการซ้อน element ไว้อยู่ชั้นบน ๆ แต่ถ้าค่าของ z-index เป็นลบก็จะเป็นการซ้อน element ไว้อยู่ชั้นล่าง ๆ

ซึ่ง element ที่สามรถใช้ z-index ได้ต้องมีการกำหนด position ตัวใดตัวหนึ่ง ดังนี้

  • position: relative || absolute || fixed || sticky ;
  • z-index: auto || number

Transform

transform คือ การที่จะใส่เอฟเฟ็กต่าง ๆ ให้แก่ element โดยมีวิธีดังนี้

  • translate() - การย้าย element ไปตำแหน่งที่กำหนด
  • rotate() - การหมุน element
  • scale() - การย่อขยาย element
  • skewX() - ทำให้เอียงในแนวนอน
  • skewY() - ทำให้เอียงในแนวตั้ง
  • matrix() - เป็นการรวม transform ทั้งหมดยกเว้น rotate เข้าด้วยกันในบรรทัดเดียว
    โดยเขียนได้ดังนี้ matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())

ข้อควรระวัง! การใช้ transform แบบผสม การเรียงลำดับ values ที่ต่างกัน จะให้ผลลัพธ์ต่างกัน

Transition

transition เป็นสิ่งที่ช่วยให้การเปลี่ยนค่าต่าง ๆ ของ element เป็นไปอย่างราบรื่น

  • transition-property - กำหนด property ที่ต้องการใช้ transition
  • transition-duration - กำหนดระยะเวลาในการ transition
  • transition-timing-function - กำหนดรูปแบบการเร่งความเร็วของ transition
  • transition-delay - กำหนดความล่าช้าของ transition
  • transition - กำหนด transition รวมกันเป็นบรรทัดเดียว
    โดยใส่ค่าตามนี้ property duration timing-function delay โดย timing-function และ delay ไม่จำเป็นต้องกำหนดก็ได้

Object-fit

object-fit คือ การกำหนดให้ < img > หรือ < video > ปรับขนาดตาม container โดยสามารถกำหนดได้ดังนี้

  • fill - เป็นค่าเริ่มต้น ภาพจะถูกปรับขนาดให้เต็มมิติที่กําหนด หากจําเป็นภาพจะถูกยืดหรือบีบให้พอดีกับกล่อง
  • contain - ภาพจะคงอัตราส่วนภาพไว้ แต่จะถูกปรับขนาดให้พอดีกับขนาดที่กําหนด เพียงด้านใดด้านหนึ่ง
  • cover - ภาพจะคงอัตราส่วนภาพไว้และเติมเต็มมิติที่กําหนด ภาพจะถูกตัดขอบออกเพื่อให้พอดีกับกล่อง
  • none - ภาพไม่ถูกปรับขนาด
  • scale-down - ภาพถูกลดขนาดลงเป็นเวอร์ชันที่เล็กที่สุดของ none หรือ contain

User-select

user-select คือ property ที่ใช้สำหรับป้องกันการ select ในบราวเซอร์ที่ใช้เอนจิ้นท์ Webkit และ Firefox และตั้งค่า cursor เป็นลูกศรแบบดีฟอลต์แทนที่จะเป็น cursor ที่ใช้สำหรับการ select เพื่อเลี่ยงการดับเบิ้ลคลิกเลือกหรือไฮไลต์ที่ข้อความได้ ส่วนใหญ่จะใช้เพื่อป้องกันการ copy ข้อความ
